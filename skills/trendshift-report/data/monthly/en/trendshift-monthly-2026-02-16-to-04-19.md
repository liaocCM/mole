**Background:** Pretext is a pure JavaScript/TypeScript library by Chenglou for multiline text measurement and layout. It sidesteps DOM measurements like `getBoundingClientRect` and `offsetHeight` — which trigger expensive layout reflows — by implementing its own text measurement logic using the browser's font engine as ground truth. The project supports rendering to DOM, Canvas, SVG, and soon server-side environments, and has gathered over 43,000 stars since its March 2026 launch.

**Problem it solves:** Browser-based text layout operations are among the most expensive DOM interactions, causing jank and performance degradation in text-heavy applications. Pretext eliminates layout reflow overhead by computing text metrics independently, enabling smooth rendering for rich text editors, canvas-based UIs, and internationalized applications that must handle complex scripts.

**Why another one?** Existing text measurement libraries either rely on DOM measurements (suffering the same performance penalty) or lack comprehensive language support. Pretext handles all scripts — including right-to-left, vertical, and complex ligature-based writing systems — while remaining a pure JS library with no native dependencies, making it uniquely portable across platforms.
