**Language:** Rust  |  **Stars:** 25789  |  **Forks:** 1578  |  **Best Score:** 1743  |  **Best Rank:** #23  |  **Days on chart:** 1/7  |  **Created:** 2025-08-04

**Background:** CC Switch by farion1231 is a Tauri-based desktop application that unifies management of five AI coding CLI tools: Claude Code, Codex, Gemini CLI, OpenCode, and OpenClaw. It provides 50+ built-in provider presets, MCP/Skills/Prompts synchronization, system tray quick-switching, and a local proxy with hot-switching and failover support. The project has reached over 25,000 stars.

**Problem it solves:** Developers using multiple AI coding tools must manually edit separate configuration files (JSON, TOML) for each tool when switching providers or updating settings. CC Switch provides visual provider switching across all five tools from a single desktop app with SQLite-backed atomic writes to prevent configuration corruption.

**Why another one?** No other tool manages all five major AI coding CLIs from a single interface. The local proxy with hot-switching and failover support adds a reliability layer, and cloud sync capabilities enable consistent configuration across multiple development machines.
